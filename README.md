# Simple Linktree Page

Sebuah halaman Linktree sederhana yang dibuat dengan HTML dan CSS dasar.

## Deskripsi
Project ini adalah halaman personal yang berisi kumpulan link repo dan biodata singkat. Dibuat sebagai latihan dasar HTML dan CSS.

## Teknologi yang Digunakan
- HTML5
- CSS3
- Font Awesome (untuk icon)

## Cara Menggunakan
1. Clone repository ini
   ```bash
   git clone https://github.com/username/Simple-Linktree-Page.git
   ```
2. Buka file `index.html` di browser atau ketik xdg-open index.html di linux

## Struktur Project
```
Simple-Linktree-Page/
├── index.html
├── style.css
├── README.md
└── assets/
    └── images/
```
